Instrucciones para correr el juego:
- Descargar todos los archivos dentro de la carpeta "Videojuego Breakout".
- Abrir el index.html en un navegador web. 
- Abrir el archivo 11, titulado Breakout

Controles:
La pala se controla con A, D:
- A = Izquierda
- D = Derecha

Objetivo del Juego:
El objetivo de este breakout es romper todos los vasos utilizando la pelota y la pala, evitando que la pelota se salga de la pantalla por la parte inferior. Al romper todos pasas de nivel.
